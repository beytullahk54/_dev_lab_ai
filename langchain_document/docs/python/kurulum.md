# Kurulum & Ortam Hazırlığı

Bu sayfa, LangGraph geliştirme ortamını hazırlamak için hızlı bir rehberdir.  
Detaylı kütüphane bilgisi için → [Kütüphaneler & venv](./kutuphaneler)

## Hızlı Başlangıç

```bash
# Python 3.10+ gerekli
python --version

# Proje kur
mkdir langgraph-proje && cd langgraph-proje
python -m venv venv
source venv/bin/activate   # Mac/Linux
pip install langgraph langchain-openai python-dotenv
```
